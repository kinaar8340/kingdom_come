# Derivation: Hopf Map via Quaternions

1. **Concept** — Unit quaternions form $S^3$; the Hopf map sends each quaternion to a point on $S^2$.
2. **Math** — $y_1 = x_1^2 - x_2^2$, $y_2 = 2x_1x_2$, $y_3 = 2(x_3x_4 + x_1x_2)$.
3. **Visualization** — Stereographic projection reveals linked Villarceau circles.
4. **Code** — `kingdom.core.hopf.sample_fiber()` and `kingdom.viz.hopf_plotly`.
5. **Physics** — Topologically protected flux tubes on the gauged lattice.
